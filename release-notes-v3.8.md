## MultiCopy v3.8

多段连续复制 + 队列按序/选择性粘贴的 Windows 桌面工具。

### 下载

| 文件 | 说明 |
| --- | --- |
| `MultiCopySetup-3.8.exe` | 安装包（推荐普通用户） |
| `MultiCopy.exe` | 绿色免安装版（单文件自包含 .NET 8 运行时，约 65MB） |

### 安装与使用

1. 双击 `MultiCopySetup-3.8.exe` 安装，或直接运行 `MultiCopy.exe`
2. 启动后常驻系统托盘，照常用 `Ctrl+C` 复制，内容自动入队
3. 按 `Alt+Z` 或点击托盘图标调出主窗口
4. 开启「按序粘贴」模式后，`Ctrl+V` 按队列顺序逐条粘贴
5. 也可单击列表项选择性粘贴、双击置顶项重复粘贴

### 主要更新

- **优化粘贴目标窗口选择**：粘贴时优先使用当前前台窗口作为目标，避免调出 MultiCopy 后切换了粘贴目标（如网页对话框）时，焦点被误切回复制源软件
- **增大剪贴板去重窗口**：文本去重时间窗口从 500ms 提升至 1500ms，减少某些应用单次 `Ctrl+C` 触发多次入队的问题
- **版本号统一**：程序集版本与安装包版本同步为 3.8

### 功能一览

- 多段连续采集：后台监听剪贴板，自动入队
- 按序粘贴：开启后 `Ctrl+V` 逐条吐出队首
- 选择性粘贴：单击列表项粘贴并出队
- 置顶常驻：双击置顶项粘贴但不出队，持久化保存
- 分组管理：QQ 风格分组，复制入组、粘贴只消耗该组
- 图片支持：采集剪贴板图片（截图、复制的图片）
- 明暗主题：一键切换
- 托盘常驻 + 全局热键：默认 `Alt+Z` 调出/隐藏
- 单实例运行
